# rocs вЂ” Roblox Open Cloud Asset Sync

Sync images, sounds, meshes, animations, and videos to Roblox via the [Open Cloud Assets API](https://create.roblox.com/docs/cloud/guides/usage-assets). Generates TypeScript asset ID files for use with [roblox-ts](https://roblox-ts.com/).

## Features

- **All asset types** вЂ” images, sounds, meshes, animations, videos
- **Hash-based change detection** вЂ” only uploads changed files (lock file tracking)
- **Recursive directory scanning** вЂ” nested folder structures become nested TypeScript objects
- **TypeScript codegen** вЂ” auto-generates `as const` asset ID files
- **Zero dependencies** вЂ” pure Node.js (>=18), no external packages
- **CLI + library** вЂ” use `npx rocs sync` or `require("roblox-open-cloud-sync")` programmatically

## Supported Formats

| Type | Extensions | Roblox assetType |
|------|-----------|-----------------|
| Image | `.png` `.jpg` `.jpeg` `.bmp` `.tga` | Decal |
| Audio | `.mp3` `.ogg` `.wav` `.flac` | Audio |
| Mesh | `.fbx` `.glb` `.gltf` `.obj` | Model |
| Animation | `.rbxm` `.rbxmx` | Animation |
| Video | `.mp4` `.mov` | Video |

## Quick Start

```bash
npm install -D roblox-open-cloud-sync
```

Create `rocs.toml`:

```toml
[creator]
type = "user"
id = 123456789

[[sync]]
name = "images"
path = "assets/images"
output = "src/shared/image-assets.ts"

[[sync]]
name = "sounds"
path = "assets/sounds"
output = "src/shared/sound-assets.ts"
```

Set your API key in `.env`:

```
ROCS_API_KEY="your-open-cloud-api-key"
```

Run:

```bash
npx rocs sync
```

## Configuration

### `rocs.toml`

```toml
[creator]
type = "user"       # "user" or "group"
id = 123456789      # Roblox User ID or Group ID

[[sync]]
name = "images"           # Group name (used for lock file: images.lock.json)
path = "assets/images"    # Directory to scan (recursive)
output = "src/shared/image-assets.ts"  # Generated TypeScript file
# assetType = "Decal"     # Optional: force asset type (auto-detected by default)
```

### Environment Variables

- `ROCS_API_KEY` вЂ” Roblox Open Cloud API key

Set in `.env` or export in your shell.

## Programmatic Usage

```javascript
const { loadConfig, loadEnv, syncAll } = require("roblox-open-cloud-sync");

loadEnv();
const config = loadConfig();
await syncAll(config, process.env.ROCS_API_KEY);
```
